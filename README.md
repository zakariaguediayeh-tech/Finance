# Mes Finances — Gestion budgétaire personnelle

PWA de gestion financière : suivi dépenses/revenus, budgets par catégorie avec alertes,
objectifs d'épargne, statistiques (donut, historique 6 mois) et section investissement éducative
avec simulateur d'intérêts composés. Données stockées localement, fonctionne hors-ligne.

## Déploiement (GitHub Pages)
1. Uploader TOUS les fichiers de ce dossier À LA RACINE du repo (pas dans un sous-dossier)
2. Settings > Pages > Source: main / root > Save
3. URL: https://USER.github.io/REPO/
4. Coller l'URL dans pwabuilder.com pour générer l'APK
